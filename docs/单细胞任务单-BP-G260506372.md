### 样品命名规则

- **报告组别名** = 实验条件 + 组织 + 生物学重复编号（如 `Ctrl_T_1` = 对照组\_肿瘤\_重复1）
  - `Ctrl` = 对照组（Control）
  - `cKO` = 条件性敲除组（conditional knockout）
  - `T` = 肿瘤组织（Tumor）
  - `L` / `Lung` = 肺组织（肺转移灶）
- **收样名称** = 送样时的原始标记（可能包含 `_` 等特殊字符）
- **报告样品名称** = 最终报告中使用的规范化名称

| 调整后字段 | 说明 |
|-----------|------|
| **过滤后细胞平均数** | 经 QC 过滤（去双细胞、去死细胞、去空液滴）后的有效细胞数均值 |
